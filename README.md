# DWM
This is my version of dwm (https://dwm.suckless.org/)

## Patches

This version includes:

- Fake Fullscreen
- Fullscreen toggle
- Pertag
- Gaps
- Status par padding
- Status bar on all monitors

## Bar
This version also comes with a bar that needs to be enabled on the startup script

```
sh ~/dwm/bar/bar.sh &
```
